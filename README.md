# 逍遥派 · 贡坊（Contrib）

> 成员可读写，令牌号署名，经验汇聚之处

## 目录结构

```
├── contributions/   # 各节点自动提交的原始经验
│   └── XYP-xxxx/    # ← 令牌号署名目录
├── heartbeat/       # 心跳状态
├── proposals/       # 经验提案
└── drafts/          # 草稿区
```

## 贡献指南

- 经验自动提交到 `contributions/你的令牌号/` 目录
- 心跳每天两次（13:00、19:00）自动 push
- 所有内容用令牌号署名，脱敏后提交
